# Security Policy

## Supported Versions
Derzeit wird die aktuelle `main`-Version aktiv gepflegt.

## Reporting a Vulnerability
Bitte melde Sicherheitsprobleme verantwortungsvoll und **nicht oeffentlich**.

Sende eine Beschreibung mit:
- betroffenen Komponenten
- reproduzierbaren Schritten
- Impact-Einschaetzung
- optionalem Fix-Vorschlag

an: `security@lifegamelab.local` (Platzhalteradresse, bei Bedarf anpassen).

## Response Ziele
- Erstreaktion: innerhalb von 72 Stunden
- Triagierung: schnellstmoeglich nach Reproduzierbarkeit
- Fix-Kommunikation: nach Absprache mit Melder:in

Danke fuer Responsible Disclosure.

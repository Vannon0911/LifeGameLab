# Support

## Wo Fragen hingehoeren
- Bug Reports: GitHub Issue mit Bug-Template
- Feature-Ideen: GitHub Issue mit Feature-Template
- Architektur-/Prozessfragen: mit Verweis auf `RUNBOOK.md` und `docs/`

## Was bei Supportanfragen helfen wird
- klare Ausgangslage und erwartetes Verhalten
- reproduzierbare Schritte
- relevante Logs / Fehlermeldungen
- Betriebssystem, Node-Version, Browser

## Reaktionshinweis
Antwortzeiten koennen variieren. Gut strukturierte Anfragen werden deutlich schneller bearbeitet.

# Cache Docs 2

Purpose
- This folder stores cache/session data that is NOT Source of Truth (SoT).
- SoT remains in canonical docs/contracts/runtime/test gates.

Rules
- Keep orchestration logs, temporary decisions, audit snapshots, and handoff receipts here.
- Do NOT treat files in this folder as authoritative architecture or contract truth.
- Update this folder after each task when non-SoT context changed.

Current policy
- Cache sync is mandatory at task start and task end.
- Cache instance refresh is mandatory after each audit.

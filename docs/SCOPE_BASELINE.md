# SCOPE_BASELINE

## Produktziel
Mobile-optimiertes Zellstrategie-Spiel auf deterministischer Simulationsbasis.

## Scope
- autonome Simulation als Regelwerk
- spielerische Eingriffe ueber Actions/Gates
- klare UX-Signale fuer Risiko, Mission, Progression

## Nicht im Scope
- Kernel-Umbau ohne zwingenden Contract-Blocker
- parallele State-Systeme
- nicht deterministische Logikpfade
